# Composition Rules

## Tarkoitus

Määrittää miten sivut rakennetaan moduleista.

## Esimerkki

City Guide:

- Hero
- Featured Events
- Category Shortcuts
- Weekend Picks
- Venue Highlights

## Säännöt

- moduulien järjestys voi vaihdella
- moduulityypit eivät saa muuttua

## Tavoite

Variaatio ilman kaaosta
