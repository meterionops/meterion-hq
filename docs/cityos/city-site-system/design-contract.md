# Design Contract

## Tarkoitus

Määrittää UI:n tekninen sopimus.

## Sisältö

- komponentit (cards, grids, hero)
- spacing system
- typography scale
- interaction rules

## Sääntö

Frontend ei saa rikkoa tätä contractia.

## Hyöty

- konsistenssi
- turvallinen generointi
