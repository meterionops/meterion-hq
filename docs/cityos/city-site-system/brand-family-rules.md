# Brand Family Rules

## Tarkoitus

Varmistaa yhtenäinen brändi kaikille kaupungeille.

## Yhteiset elementit

- typography
- spacing
- component system
- layout grid

## Vaihtelevat elementit

- color palette
- imagery
- tone

## Sääntö

Kaikkien kaupunkien pitää tuntua samalta tuotteelta  
→ vaikka ne ovat visuaalisesti erilaisia
