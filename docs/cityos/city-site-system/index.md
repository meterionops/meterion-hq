# City Site System v1

City Site System määrittelee miten CityOS rakentaa kaupunkisivut.

## Tavoite

- Yhteinen rakenne kaikille kaupungeille
- Ohjattu visuaalinen variaatio
- Pertti voi generoida uusia kaupunkeja automaattisesti

## Periaate

Structure is fixed  
Variation is guided  
Generation is automated
